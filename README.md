# Forkfolio
A web application that allows users to look through a catalog of recipes that they can also comment on and give ratings to. Users will also be able to submit their own recipes that an admin can log in and review and decide whether to add them to the catalog 


# How to run
- Open Front_end and Back_end folders in two terminals. 
- Run `npm i` in both folders to install the node modules.
- Run `npm start` in both folders to start the server.
- Go to your Front_end terminal and `CTRL + CLICK` the link, or go to your web browser and visit `http://localhost:7000`
